# BI Analyst Internship at Aspex  
**4-недельная стажировка | Июнь 2025**

## О проекте
Прошла 4-недельную стажировку в роли **BI-аналитика** в компании Aspex.  
С нуля освоила прототипирование дашбордов в **Miro** и участвовала в оцифровке производственных и проектных процессов.

Проект состоял из двух частей:
- Индивидуальной (оцифровка завода по книге «Цель»)
- Командной (проект «Agile Insight» — умные решения для жёстких дедлайнов)

## Что было сделано

### Недели 1–2 | Индивидуальная работа
- Оцифровала производственный процесс завода на основе теории ограничений (книга **«Цель»** Голдратта).
- Разработала систему метрик для выявления узких мест.
- Прототипировала **3 страницы дашборда** в Miro:
  - Детектор узких мест
  - Паспорт узкого места
  - Эффективность завода

### Недели 3–4 | Командная работа
- Работала в кросс-функциональной команде (PM, BA, BI-аналитик).
- Провела discovery компании по принципам книги **«Deadline»**.
- Выявила ключевые бизнес-проблемы и предложила решение в формате дашбордов.
- На этапе delivery прототипировала **3 дашбордов**, разделённых на 3 блока:
  - **Стратегический**
  - **Тактический**
  - **Операционный**
